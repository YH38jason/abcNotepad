# Python-文本编辑器（Wx）
一个十分普通的文本编辑器

## 前置条件
* wxPython
* python 3.10
* 任意操作系统

## 使用方法
运行main.py即可

## 前置条件安装

`pip install wxpython`

## 添加语言
### 1、在strings.json中添加相应的字符串
### 2、在lang_config/languages中添加语言的缩写（如jp、en）
### 3、在lang_config中添加语言的全称（如"English":"en"）
